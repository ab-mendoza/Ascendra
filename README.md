# Ascendra

Ascendra is a full-stack Sales Performance Platform designed to help organizations securely register, manage, and analyze sales data.

This project was built as part of my journey toward becoming a Full Stack Developer and Data Scientist, with the goal of combining software engineering, cloud deployment, and machine learning into a single business application.

---

## Features

- Secure user authentication
- JWT-based authorization
- Password hashing with BCrypt
- Sales registration
- PostgreSQL database integration
- REST API built with ASP.NET Core
- Responsive web interface
- Role-ready architecture for future expansion
